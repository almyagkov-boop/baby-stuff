# 🧸 Baby Stuff

Baby Stuff — это веб-приложение для управления списком вещей к рождению ребенка.

Источник данных — Google Sheets (CSV).

## Возможности

- 📦 Красивые карточки товаров
- 🔍 Поиск
- 🏷️ Фильтрация
- ⭐ Важность
- 🚦 Статусы
- 💶 Стоимость
- 🔗 Ссылки
- 🖼️ Предпросмотр товаров
- 🌙 Темная тема

## Стек

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React
- PapaParse

## Запуск

```bash
npm install
npm run dev
```
